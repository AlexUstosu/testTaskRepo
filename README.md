# Тестовое задание - кросплатформенное русcкоязычное приложение на Qt
Данное приложение разработано как решение тестового задания по созданию кросплатформенного приложения с графическим интерфейсом на С++ с использование фреймворка Qt. Представленная экранная форма полностью на русском языке и соответсвует требованиям задания.

## Основной функционал
Приложение позволяет: 
* добавлять геометрические фигуры в список - круги и прямоугольники;
* указывать праметры фигур:
  * для круга - радиус;
  * для прямоугольника - ширина и высота.
* просматривать список добавленных фигур;
* выбирать фигуру из списка и просматривать её площадь.

## Интерфейс
Главная экранная форма приложения включает:
* выпадающий список доступных фигур;
* набор полей для ввода параметров, изменяющихся в зависимости от выбранной фигуры;
* список фигур, отображающий добавленные объекты;
* виджеты для отображения информации о площади.

## Архитектура
Shape - абстрактный базовый класс, содержащий виртуальные методы:
1. type() - возвращает тип объекта.
2. parameters() - возвращает коллекцию параметров фигуры.
3. area() - подсчитывает площадь.

Circle, Rectangle - наследники Shape. Реализуют виртуальные методы родителя, с учетом типа фигуры.
ShapeListModel - класс модели фигуры для поддержки паттерна MV и возможности организовывать объекты в список.
MainWindow - класс главного окна приложения.

## Контрольные вопросы
***1. Насколько архитектура приложения масштабируема под добавление новых типов фигуры, параметров фигуры, новых типов запрашиваемых характеристик из списка.***

Для добавления новых фигур был разработан абстрактный класс Shape, который требует наследования нового типа фигур и реализации его методов. 
Модель представления подхватит новый тип фигуры без изменения кода. На форме изменяется только список ComboBox и набор параметров.
QVariant позволяет добавлять новые параметры довольно легко.
